## Minesweeper -- [[live]](https://isimeri.github.io/MineSweeper/)

The timeless, classic game of Minesweeper.  
In order to win, you must either mark all the bombs, or uncover all the cells that are not bombs.  
Use numbers on the board to figure out where the bombs are. Each number represents the number of bombs that are present in the neighbour cells around it.  
For example, a cell containing number **2** means that **2** out of its 8 neighbours (**5** neighbours in case of an **edge** cell, **3** neigbours in case of a **corner** cell) are bombs.
